## A. Landement du projet

Pour lancer l'application, il faut faire les instructions suivantes : 
```
Dans un terminal dans le répertoire du projet :
    $ npm install
    $ nmp run watch
    $ npx serve -s -l 8000

Dans un navigateur : 
    http://localhost:8000
```
